# AuditCore

## Descripción

AuditCore es una plataforma de evaluación de controles de seguridad y auditoría diseñada para automatizar la validación de estándares como:

- CIS Benchmarks
- NIST
- ISO 27001
- PCI DSS
- SWIFT
- DORA

La plataforma recopila evidencias técnicas, evalúa controles automáticamente y genera reportes ejecutivos y técnicos.

---

## Objetivos

- Automatizar auditorías.
- Generar evidencia automáticamente.
- Reducir el tiempo de evaluación.
- Generar reportes HTML, PDF y Excel.
- Permitir agregar nuevos módulos sin modificar el núcleo.
